# Entropy
[//]: # $$S = f(U,V)$$
$$dS = \frac{1}{T}dU + \frac{P}{T}dV$$
$$\int dS = S = \frac{U}{T} + \frac{PV}{T}$$
# Internal Energy
$$U = f(S,V)$$
$$dU = TdS - PdV$$
$$\int dU = U = TS - PV$$
# Enthalpy
$$H = f(S,P)$$
$$dH = TdS + VdP$$
$$\int dH = H = TS + PV$$
# Helmholtz Free Energy
$$A = f(T,V)$$
$$dA = -SdT - PdV$$
$$\int dA = A = U - PV$$
# Gibbs Free Energy
$$G = f(T,P)$$
$$dG = -SdT + VdP$$
$$\int dG = G = U + PV - TS$$
